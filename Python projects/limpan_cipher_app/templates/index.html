About Limpan Cipher

Encoding Rules:

  • Vowels are replaced with specific patterns:
    • 'a' → 'im'
    • 'e' → 'pa'
    • 'i' → 'li'
    • 'o' → 'mo'
    • 'u' → 'nu'
  • Consonants have 'an' appended to them (e.g., 'b' → 'ban', 't' → 'tan')
  • Non-alphabetic characters remain unchanged
  • Capitalization is preserved in the transformation

Examples:

English Limpan Code
Hello Han-palan-lan-lan-mo
Limpan Lan-li-man-pan-im-nan
Code Can-mo-dan-pa

Command Line Usage:

This cipher can also be used from the command line:

# Encode text
python limpan_cipher.py encode "Hello World"

# Decode text
python limpan_cipher.py decode "Han-palan-lan-mo Wan-mo-ran-lan-dan"
